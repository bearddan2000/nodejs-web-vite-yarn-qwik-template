# nodejs-web-vite-yarn-qwik-template

## Description
A qwik template for the vite packager.

## Tech stack
- yarn
- vite
- qwik

## Docker stack
- node

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
